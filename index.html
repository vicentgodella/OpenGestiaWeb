Open Gestia

Open Gestia es una aplicación en entorno web para gestionar asociaciones.

Creada a partir de las necesidades de la asociación juvenil Juniors M.D.

Actualmente se encuentra en fase de desarrollo.

Metodologías Ágiles

Este proyecto está siendo construido con metodologías ágiles, y definiendo la funcionalidad a partir de las historias de usuario que se nos proponen desde Juniors M.D.

Testing

Para su desarrollo, nos estamos basamos en BDD, TDD, tests unitarios, integración continua, ...

Desarrollo en Symfony 2

Este proyecto está siendo desarrollado en PHP, concretamente utilizando Symfony 2.1.
